==tile==
Contains the tile (OpenMapquest Style) under the CC-By-CA Creative Commons Licence from OpenStreetMap and Contributors Data

You can build the tiles by:
 
-Maperitive 
http://maperitive.net/

-TileMill
http://mapbox.com/tilemill/

-Mapnik
http://mapnik.org/

Or grab with tools like MOBAC:
http://mobac.sourceforge.net/

==leaflet==
Leaflet is a modern open-source JavaScript library for mobile-friendly interactive maps.
 It is developed by Vladimir Agafonkin with a team of dedicated contributors.
 Weighing just about 28 KB of JS code, it has all the features most developers ever need for online (And in PirateBox offline ;) maps.
